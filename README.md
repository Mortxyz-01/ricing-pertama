```bash
1. Window Manager & Tampilan
Install Hyprland, Waybar, Rofi, dan komponen visual lainnya.

yay -S hyprland waybar rofi-wayland wlogout swayosd-git nwg-look swww

2. Terminal & Shell
Terminal emulator dan pernak-pernik shell (Fish, Starship, dll)
yay -S kitty fish starship fastfetch btop cava

3. Fonts & Icons wajib derr
Jangan dilewatkan, nanti icon wifi/baterai jadi kotak-kotak (tofu) kalau ini tidak diinstall
yay -S ttf-jetbrains-mono-nerd ttf-font-awesome papirus-icon-theme

4. Utilities (Screenshot, Audio, Brightness)
yay -S grim slurp wl-clipboard pavucontrol brightnessctl playerctl

5. ASUS TUF/ROG
Install kalau laptop lu ROG/TUFF
yay -S asusctl supergfxctl rog-control-center
